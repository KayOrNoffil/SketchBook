SketchBook — это интуитивно понятное приложение для рисования и создания цифровых произведений искусства, разработанное с использованием библиотеки PyQt.
Проект ориентирован на художников, дизайнеров и любителей рисования, предоставляя мощные инструменты для творчества в удобном и доступном формате.

Техническое задание
Цели и задачи проекта:
Разработать графическое приложение для рисования с использованием PyQt5.
Реализовать функционал для рисования на холсте с возможностью выбора цвета, размера кисти и типа кисти.
Обеспечить возможность рисования различных форм (прямоугольники, круги, треугольники) и свободного рисования.
Реализовать функции сохранения и открытия изображений.
Создать базу данных для хранения истории действий пользователя.
Реализовать возможность отмены последнего действия.
Функциональные требования:
Интерфейс пользователя:

Главное окно с панелью инструментов.
Поля для ввода размера кисти и выбора типа кисти.
Выпадающие списки для выбора типа формы и цвета.
Инструменты рисования:

Свободное рисование.
Рисование прямоугольников, кругов и треугольников.
Выбор цвета и размера кисти.
Работа с изображениями:

Сохранение текущего изображения в формате PNG.
Открытие существующих изображений.
История действий:

Запись всех действий пользователя в базу данных.
Отображение истории действий в виде всплывающего окна.
Возможность отмены последнего действия.
База данных:

Использование SQLite для хранения истории действий.
Таблица с полями: id, action, timestamp.
Нефункциональные требования:
Приложение должно быть кроссплатформенным и работать на Windows, macOS и Linux.
Интерфейс должен быть интуитивно понятным и удобным для пользователя.
Приложение должно быть устойчивым к ошибкам и обеспечивать корректное выполнение всех функций.
Пояснительная записка
Данный проект представляет собой приложение для рисования, которое позволяет пользователям создавать и редактировать изображения с помощью различных инструментов. Приложение разработано с использованием библиотеки PyQt5, что обеспечивает кроссплатформенность и современный интерфейс.

Основные компоненты приложения включают:

Класс Database: отвечает за взаимодействие с базой данных SQLite, включая создание таблицы, добавление действий, получение истории и удаление последнего действия.
Класс CanvasSizeDialog: диалоговое окно для установки размера холста.
Класс PaintApp: основной класс приложения, который управляет интерфейсом, обработкой событий мыши и рисованием на холсте.
Приложение позволяет пользователям:

Устанавливать размер холста перед началом рисования.
Выбирать цвет и размер кисти, а также тип кисти (обычная, круглая, квадратная, ластик).
Рисовать свободно или выбирать различные формы (прямоугольники, круги, треугольники).
Сохранять и открывать изображения в формате PNG.
Просматривать историю своих действий и отменять последние действия.
В результате, приложение предоставляет пользователям мощный инструмент для творчества и самовыражения, а также удобный способ отслеживания и управления своими действиями.
